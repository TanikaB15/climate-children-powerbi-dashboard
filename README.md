# How Climate Impacts Children Power BI Dashboard

## Project Purpose
This dashboard explores how climate hazards may interact with food and water stress to affect child health, nutrition, survival, and protection outcomes.

## Dashboard Pages
1. Overview
2. Exposure Layer
3. Vulnerability Layer
4. Child Outcomes
5. Causal Chain
6. AI Assist
7. Methodology

## Indicators
Exposure Layer:
- Floods
- Temperature anomalies
- Rainfall changes

Vulnerability Layer:
- Food insecurity
- Undernourishment
- Unsafe water exposure

Child Outcome Layer:
- Under-5 mortality
- Infant mortality
- Neonatal mortality
- Child stunting
- Child labour

## Data
This first version uses simplified sample data for learning and dashboard design.

## Important Note
This dashboard shows associations and risk pathways. It does not prove direct causation.
